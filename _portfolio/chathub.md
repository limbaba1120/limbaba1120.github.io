## 한 줄 소개

클라우드 기반 실시간 화상채팅이 가능한 개발자 커피챗 서비스입니다. 기획·개발·운영을 팀원들과 협업하며 진행했습니다.

## 팀 구성

- 백엔드 2명 / 프론트엔드 1명
- 협업 도구: GitHub · Discord

## 내가 기여한 부분

### 1. 실시간 화상 채팅 기능 구현

- **Amazon Chime SDK**를 활용해 사용자 간 실시간 오디오/비디오 송수신이 가능한 화상 채팅 기능을 구현하여 고품질의 화상 채팅 환경 제공
- AWS Chime SDK의 **Meeting / Attendee / Voice Connector API** 를 사용해 화상 채팅 세션을 생성하고 관리하는 백엔드 로직 개발

### 2. 소셜 로그인 (Spring Security + OAuth 2.0)

- 개발자들에게 친숙한 **GitHub OAuth**를 활용해 회원가입 및 로그인 절차 간소화
- Spring Security 프레임워크와 OAuth 2.0 클라이언트 라이브러리를 활용

## 느낀 점

프론트엔드 팀원과 처음으로 협업한 프로젝트입니다. Spring Security와 WebRTC 같은 기술을 다루는 데 있어 처음에는 많은 어려움을 느꼈습니다.

백엔드 개발 경험이 부족해서 기술을 깊이 이해하는 데 한계를 느꼈지만, 강의를 듣고 영어로 작성된 공식 문서를 읽으며 Chime SDK를 학습했고, 기술에 대한 자신감이 생겼습니다.

단순히 강의만 보며 따라 하던 과거와는 다르게, 프로젝트에서 기술을 사용해보고 문제를 해결했던 경험이 개발 실력을 빠르게 향상시켜주었고, 개발에 대한 즐거움이 생겼습니다.

앞으로는 다른 프로젝트를 하면서 다양한 사람들과 협업을 더 많이 해야겠다고 생각했고, 새로운 기술을 배우면서 더 깊이 이해하고 빠르게 성장하는 개발자가 되고 싶다고 느꼈습니다.
